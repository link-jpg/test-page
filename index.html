Comando Usado en la parte de git

  1. *ls* -> muestra una lista de directorios
  2. *cd* -> nos trasladamos a la carpeta especifica
  3. *cd ..* -> lo usamos para retroceder
  4. *clear* -> lo usamos para borrar pantalla
  5. *pwd* -> muestra la ruta actual
  6. *mkdir* -> crea el directorio --carpeta
  7. *mr* -> borrar mas el nombre del doc
  8. *git init* para iniciar git
  9. *git add* y usamos el . para hacerlo global en la carpeta
  10. *git commit -m* agramos comillas "proyecto version 1" despues se hara correctamente
  11. *git branch -M* y colocamos el nombre
