Comando Usado en la parte de git

  • *ls* -> muestra una lista de directorios
  • *cd* -> nos trasladamos a la carpeta especifica
  • *cd ..* -> lo usamos para retroceder
  • *clear* -> lo usamos para borrar pantalla
  • *pwd* -> muestra la ruta actual
  • *mk* -> muestra la ruta actual
  • git init
  • *git add* y usamos el . para hacerlo global en la carpeta
  • git commit
